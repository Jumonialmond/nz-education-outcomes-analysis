# NZ Education Outcomes Analysis

Exploratory analysis of New Zealand school leaver attainment trends by region and ethnicity using Ministry of Education data (2014-2024).

**Status:** Work in progress

**Tools:** Python, pandas, matplotlib, seaborn

**Data source:** Ministry of Education - Education Counts (https://www.educationcounts.govt.nz)

## Key Questions
- How has NCEA Level 2 achievement varied by region over the past decade?
- Which regions show the largest gaps in educational attainment?
- How did COVID-19 (2020-2021) affect achievement rates across NZ?
